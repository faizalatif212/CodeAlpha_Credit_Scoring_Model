# CodeAlpha_Credit_Scoring_Model
Task 1: Credit Scoring Model built during Machine Learning Internship at CodeAlpha

## 📌 Project Overview
This project focuses on building a robust Credit Scoring Model to predict the creditworthiness of individuals. The primary objective is to classify historical customer data to determine the likelihood of default, helping financial institutions make data-driven lending decisions and minimize financial risk.

## 🛠️ Tech Stack & Key Concepts
* **Language:** Python
* **Libraries:** Pandas, NumPy, Scikit-Learn, Imbalanced-Learn
* **Data Preprocessing:** Handle missing values, One-Hot Encoding for categorical features.
* **Class Imbalance:** Applied SMOTE (Synthetic Minority Over-sampling Technique) to balance the target variable.
* **Machine Learning Algorithm:** Random Forest Classifier

## 📊 Model Evaluation & Results
The model was evaluated using standard metrics to ensure high accuracy and balance:
* **Precision & Recall:** Optimized to effectively identify high-risk (Bad) customers.
* **Feature Importance:** Analyzed key drivers affecting credit scores (e.g., income, credit history, and age).

## 📂 How to Run
1. Open the `.ipynb` notebook in Google Colab or Jupyter Notebook.
2. Run the code cells sequentially to process the data, train the model, and view evaluation metrics.
